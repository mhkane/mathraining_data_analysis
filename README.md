# Mathraining country data analysis

Tools to scrape [Mathraining](https://www.mathraining.be/) (French IMO training site), build yearly metrics and gender breakdowns, and produce report figures and CSVs.

**[Run in Colab](https://colab.research.google.com/drive/1twcJVzNbkOCum3YyH1y_PImMLpRYaotD#scrollTo=HPS2yD0fLSB-)** — clone repo, install deps, run pipeline (no local setup).

## About Mathraining

[Mathraining](https://www.mathraining.be/) launched in December 2014. It offers theory (combinatorics, geometry, number theory, algebra, etc.), problems by topic, virtual tests, and contests. The site reports tens of thousands of ranked users and over a million exercises solved; users come from Belgium, France, and many francophone countries (e.g. Morocco, Tunisia, Côte d'Ivoire, Algeria).

## What this repo does

- Scrapes country user lists and resolution history from Mathraining.
- Builds yearly/cumulative metrics (accounts, points, exercises, problems) overall and by gender.
- Produces bar charts, gender pie charts, and “top N” rankings.
- Saves/loads data via pickle so you can re-run analysis without re-scraping.

## Project structure

| Item | Description |
|------|-------------|
| `mathraining_scraping_utils.py` | Scraping, metrics, plotting, report build. |
| `Mathraining_Country_Report.ipynb` | Notebook: Part 1 = run pipeline & save pickle; Part 2 = load & analyze. |
| `report/` | Outputs: see `report/README.md`. March 6th 2026 report has `csvs/` (6 CSVs) and `figures/` (14 PNGs). |
| `scripts/export_report_outputs.py` | Export report CSVs and PNGs from pickle or from a fresh scrape (`--build`). |

`report_data.pkl` is generated by the notebook and ignored by git.

## Setup and usage

```bash
pip install requests beautifulsoup4 pandas matplotlib tqdm
```

1. Open `Mathraining_Country_Report.ipynb`, set `country_id` (e.g. `48` for Côte d'Ivoire).
2. **Part 1:** Uncomment and run the pipeline cell; optionally save with `msu.save_report_data(report, "report_data.pkl")`.
3. **Part 2:** Load pickle and run `msu.build_report_from_data(data)` to get figures and tables.

To export the March 6th 2026 report: `python scripts/export_report_outputs.py` (from pickle) or `python scripts/export_report_outputs.py --build` (full scrape).

Use moderate `sleep_seconds` between requests when scraping.
